# Random Task Generator

**Автор:** Калинин Александр

## Описание

Random Task Generator — это графическое приложение на Python для генерации случайных задач по категориям. Оно помогает разнообразить день, предлагая неожиданные дела из учёбы, спорта или работы. Все сгенерированные задачи сохраняются в историю (JSON) и могут быть отфильтрованы по типу.

## Возможности

- Генерация случайной задачи из предопределённого списка + добавленных пользователем
- Фильтрация по категориям: учёба, спорт, работа
- Добавление своих задач с указанием категории
- История всех сгенерированных задач (сохраняется в `tasks.json`)
- Очистка истории
- Валидация ввода (пустые строки не допускаются)
